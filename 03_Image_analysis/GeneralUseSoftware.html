Uso general de software#

Las herramientas en esta página tienden a ser relativamente multipropósito en muchos tipos de análisis y/o imágenes. Para herramientas que se especializan en ciertos pasos de análisis, consulte la página [Software de uso específico] (./SpecificUseSoftware.md).

ImageJ#

ImageJ 27,28,29 es un programa de procesamiento de imágenes que es capaz de operar en una variedad de imágenes que incluyen multicanal, y series de 3D y de tiempo. Proporciona una variedad de operaciones básicas de procesamiento de imágenes, pero se puede complementar con una variedad de plugins para tareas más complejas.

¿Cuáles son sus desventajas?

  • Las operaciones de procesamiento de imágenes se realizan una a la vez, aunque tiene la capacidad de procesamiento por lotes y la creación de macros, requiere cierta comprensión de programación.

  • Pero si bien puede abrir imágenes grandes, existe un límite de tamaño para el tamaño que puede manejar según la memoria disponible. E incluso si puede abrir imágenes grandes, puede desacelerar el rendimiento.

¿Cómo descargar/instalar y obtener más información?

Para descargar ImageJ o su distribución de «baterías incluidas», Fiji, vaya a

  • 🌐 [Descarga de ImageJ] (https://imagej.net/downloads)

Para obtener documentación y tutoriales sobre cómo usar ImageJ, así como una lista de plugins disponibles

  • 🌐 [Conceptos básicos de ImageJ] (https://imagej.net/learn/)

CellProfiler#

CellProfiler 30 es un software diseñado por biólogos para biólogos; crea un puente entre el análisis de imágenes y el científico sin necesidad de conocimientos informáticos. Fue diseñado con la idea de una canalización de análisis de imágenes en mente: le permite tomar una serie de módulos interoperables para diseñar su propia canalización de análisis personalizada que se puede aplicar a una o miles de imágenes, haciéndola adecuada para el análisis de imágenes de alta producción (high-throughput).

¿Cuáles son sus desventajas?

  • CellProfiler no puede manejar imágenes grandes, como secciones de tejido completo de experimentos histológicos. El tamaño de la imagen está actualmente limitado por la memoria disponible en su computadora.

  • Si bien CellProfiler puede realizar análisis en imágenes 3D, la visualización se limita a un plano z a la vez a través de un control deslizante en la ventana de visualización.

  • Además, varias funciones de CellProfiler solo están disponibles para imágenes 2D.

QuPath#

QuPath 31 ofrece un amplio conjunto de herramientas de análisis de imágenes que se pueden aplicar a imágenes de diapositivas completas. Por ese motivo, se utiliza ampliamente con imágenes de patología, pero también se puede utilizar con otras imágenes. QuPath también contiene herramientas de clasificación de píxeles y puede integrarse con ImageJ (por ejemplo, para enviar ROI entre los programas o para acceder a complementos de ImageJ).

¿Cuáles son sus desventajas?

  • Para sacar el máximo provecho de QuPath (especialmente cuando se analizan muchas imágenes), se necesitan algunos scripts y conocimientos de programación (o la adaptación del código de otros).

¿Cómo descargar/instalar y obtener más información?

Icy#

Icy 32 es una herramienta de análisis de imágenes lista para usar, utiliza complementos para crear protocolos de análisis de imágenes visuales que se pueden compartir con otros usuarios.

¿Cuáles son sus desventajas?

  • La interoperabilidad de Icy con otros softwares está limitada a ImageJ

¿Cómo descargar/instalar y obtener más información?

Para descargar Icy

  • 🌐 [descarga Icy] (https://icy.bioimageanalysis.org/download/)

Para ejemplos de documentación y tutoriales.

  • 🌐 Curso y tutorial de Icy

  • 🎥 Análisis de bioimagen con Icy

MIB (Explorador de imágenes de microscopía)#

MIB 33 es un software fácil de usar para el análisis de imágenes de conjuntos de datos multidimensionales para microscopía óptica y electrónica. Le permite utilizar todos los datos adquiridos para su análisis y la extracción de características morfológicas.

¿Cuáles son sus desventajas?

  • Fue creado utilizando MATLAB, existe una versión empaquetada independientemente, pero no utilizan las versiones más actualizadas de MATLAB

¿Cómo descargar/instalar y obtener más información?

Para descargar MIB

  • 🌐 [descarga de MIB] (http://mib.helsinki.fi/downloads.html)

Para ejemplos de documentación y tutoriales.

  • 🌐 [Guía de usuario de MIB] (http://mib.helsinki.fi/help/main2/im_browser_user_guide.html)

  • 🌐 Tutoriales

napari#

napari 34 se está desarrollando como un visualizador de imágenes multidimensional que se puede expandir a través de una variedad de plugins para realizar tareas de análisis de imágenes básicas y complejas.

¿Cuáles son sus desventajas?

  • napari aún se encuentra en las etapas de desarrollo, pero es una plataforma muy popular y ya cuenta con una variedad de plugins y casos de uso con tutoriales disponibles.
