Software de código abierto#

Introducción#

Cuando llega el momento de seleccionar un programa de software para el análisis de imágenes, hay muchas opciones, algunas más generales y otras altamente especializadas en modalidades de imágenes o tipos de experimentos específicos. En general, un buen lugar para comenzar a explorar es examinar documentos en su campo y ver lo que otros han usado para analizar experimentos similares al suyo. Es importante tener en cuenta que no hay una respuesta correcta a «¿Qué programa debo usar?» Dependiendo de su pregunta biológica, sus imágenes y su propia comodidad con la programación, hay muchas opciones disponibles.

A continuación, resumimos los casos de uso y las limitaciones de algunos de los software más comunes, gratuitos y de código abierto para el análisis de imágenes. Esta es una pequeña lista y existen otras más extensas como la Guía del autoestopista a través del software de análisis de bioimágenes universo 3 y el BioImage Informatics Index { cite}Paul-Gilloteaux2021-vw.

Cualquiera que sea el software que elija, asegúrese de incluir una descripción detallada de su análisis en sus métodos, con archivos de canalización o de flujo de trabajo si es posible, para que otros puedan reproducir su trabajo. ¡También asegúrese de citar el software de análisis que utiliza! Esto ayuda a los desarrolladores del software a obtener subvenciones y ayuda a otros a encontrar herramientas útiles.

ImageJ

ImageJ es un programa de procesamiento de imágenes que es capaz de operar en una variedad de imágenes que incluyen multicanal, y series 3D y temporales. Proporciona operaciones básicas de procesamiento de imágenes y tiene una variedad de plugins para tareas más complejas. Leer más…

CellProfiler

CellProfiler se diseñó con la idea de una línea de análisis de imágenes en mente; le permite tomar una serie de módulos interoperables para diseñar su propia línea de análisis personalizada que se puede aplicar a una o miles de imágenes, lo que lo hace adecuado para el análisis de imágenes de alta producción (high-throughput). Leer más…

QuPath

QuPath ofrece un amplio conjunto de herramientas de análisis de imágenes que se pueden aplicar a imágenes de diapositivas completas, como imágenes de patología, pero también se pueden usar con otras imágenes. QuPath también contiene herramientas de clasificación de píxeles y puede integrarse con ImageJ. Leer más…

Icy

Icy es una herramienta de análisis de imágenes lista para usar, utiliza complementos para crear protocolos de análisis de imágenes visuales que se pueden compartir con otros usuarios. Leer más…

MIB

MIB es un software fácil de usar para el análisis de imágenes de conjuntos de datos multidimensionales para microscopía óptica y electrónica. Le permite utilizar todos los datos adquiridos para su análisis y extracción de características morfológicas. Leer más…

napari

napari se está desarrollando como un visualizador de imágenes multidimensionales que se puede expandir a través de una variedad de plugins para realizar tareas de análisis de imágenes básicas y complejas. Leer más…

Cellpose

Cellpose es un algoritmo de segmentación , proporciona una interfaz gráfica de usuario que permite a los usuarios usar modelos entrenados o entrenar los suyos propios usando sus imágenes y anotaciones. Leer más…

ilastik

ilastik es una herramienta para la clasificación, segmentación y análisis de imágenes interactivas. Aprovecha los algoritmos de aprendizaje automático para realizar una clasificación a nivel de píxeles y objetos. Usarlo no requiere experiencia en procesamiento de imágenes. Leer más…

Piximi

Piximi es una aplicación de anotación y clasificación que se ejecuta completamente desde su navegador y no requiere instalación y requiere una configuración mínima. Leer más…
